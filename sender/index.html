Chromecast Hub Server

Small proof of concept for future android homescreen app feature.

Unlike chromes inbuilt tab casting, which works in practice as screen sharing and would require this device to remain active and continously send a video feed, this approach utilizes the smart devices inbuilt web renderer. This means that the content is actutually rendered in the devices inbuilt webbroser.

This service simply tells the local chromecast which service to run from the web, but also allows for communication between this window and the one rendering on the chromecast.
